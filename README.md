# 🪙 Binance Balance Tracker (Баланс Трекер)

Красивое, быстрое и современное кроссплатформенное десктопное приложение на Python с использованием библиотеки **Flet** (Flutter для Python). Приложение позволяет в реальном времени отслеживать общую стоимость ваших активов на бирже Binance, суммируя балансы с кошельков **Spot** (Спотовый) и **Funding** (Пополнение) и переводя все монеты в эквивалент USD по текущему курсу.

---

## ✨ Основные возможности

*   **⚡ Удобный дашборд:** Отображение общего баланса в USD на большой градиентной золотой карте.
*   **💼 Объединение кошельков:** Поддержка балансов как спотового кошелька (Spot), так и кошелька пополнения (Funding).
*   **💵 Конвертация в реальном времени:** Автоматический расчет стоимости всех токенов в USD по актуальным ценам API Binance.
*   **🔍 Умная фильтрация и поиск:** Мгновенный поиск монет по тикеру и переключатель для скрытия мелких балансов (пыли) меньше $1.
*   **🛡️ Безопасность на первом месте:**
    *   API-ключи хранятся исключительно локально на вашем устройстве в зашифрованном хранилище Flet (Client Storage).
    *   Приложению требуются права **«Только чтение»** (Read Info). **Никогда не включайте** права на вывод средств (Withdrawal) или торговлю (Trade).
*   **🎨 Премиум дизайн:** Адаптивный темный интерфейс в фирменных цветах Binance с эффектами стекломорфизма, плавными микро-анимациями при наведении и прогресс-барами распределения портфеля.

## 🔑 Инструкция по созданию API ключей Binance

Чтобы приложение могло видеть ваши балансы, необходимо предоставить ему доступ:

1. Войдите в свой аккаунт на [Binance](https://www.binance.com).
2. Перейдите в меню профиля -> **Управление API** (API Management).
3. Нажмите **Создать API** (Create API) -> Выберите "Сгенерированный системой" (System generated).
4. Задайте имя (например, `FletBalanceTracker`).
5. Пройдите проверку безопасности.
6. **ОЧЕНЬ ВАЖНО:** В ограничениях API (API restrictions) должно быть включено **только**:
   *   `Включить чтение` (Enable Reading) — обычно включено по умолчанию.
   *   Все остальные галочки (например, «Включить спотовую и маржинальную торговлю», «Включить возможность вывода средств» и т.д.) **должны быть выключены**!
7. Скопируйте **API Key** и **API Secret**.
8. Откройте приложение Binance Balance Tracker, нажмите на иконку шестеренки (настройки) в правом верхнем углу, вставьте ключи и нажмите **Сохранить**.

## 🎨 Технические особенности реализации

*   **Синхронизация времени:** Binance API строго отклоняет запросы с рассинхронизацией времени компьютера более чем на несколько секунд. В `binance_client.py` встроена автоматическая калибровка смещения времени локального компьютера относительно серверов Binance (`time_offset`), что предотвращает распространенные ошибки авторизации.
*   **Фоновое обновление (Threading):** Запросы к сети происходят в отдельном потоке. Это сохраняет интерфейс полностью отзывчивым, без зависаний окон во время загрузки данных.
*   **Умная обработка прав:** Если ваш API-ключ имеет доступ к спотовому кошельку, но не имеет доступа к кошельку пополнения (Funding), приложение не упадет с ошибкой — оно отобразит баланс спота и выдаст мягкое предупреждение о необходимости проверить права Funding в настройках аккаунта Binance.
