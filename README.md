# 🪙 Binance Portfolio Tracker & Widget (Баланс Трекер)

Красивое, быстрое и современное веб-приложение на **Python (Flask)** и **Vanilla JS**, разработанное для мониторинга активов на бирже Binance в реальном времени. Оно объединяет все балансы из различных разделов биржи и выводит их в едином элегантном дашборде с возможностью мгновенного перевода в фиат (UAH, EUR, USD, RUB, BTC).

Приложение спроектировано как в виде полноценного веб-интерфейса, так и в виде компактного, готового к встраиванию интерактивного виджета.

---

## ✨ Основные возможности

*   **💼 Полная агрегация активов:** Автоматический сбор и объединение балансов из разделов:
    *   **Spot (Спотовый кошелек)**
    *   **Funding (Пополнение)**
    *   **Simple Earn (Гибкие и фиксированные депозиты)**
    *   **Trading Bots (Торговые боты)**
    *   **Futures (Фьючерсы COIN-M и USDⓈ-M)**
*   **💵 Конвертация в реальном времени:**
    *   Автоматический перевод всех токенов в эквивалент **USD** по актуальным котировкам Binance API.
    *   Быстрая конвертация общего баланса в **UAH, EUR, RUB** или **BTC** с использованием официальных банковских курсов в реальном времени через внешние API.
    *   Поддержка цепочек конверсии для редких токенов (через BTC/ETH), если прямая пара к USDT отсутствует.
*   **🎨 Премиум-дизайн:**
    *   Темный адаптивный интерфейс с использованием эффекта матового стекла (**Glassmorphism**).
    *   Фирменная палитра Binance (золотые градиенты, акцентные индикаторы тренда, плавная анимация заблика на картах и ховер-эффекты).
    *   Микро-анимация обновления данных и интерактивные статусы кошельков.
*   **🛡️ Максимальная безопасность (Два режима работы):**
    *   **Клиентский режим (по умолчанию):** Ключи API вводятся в настройках браузера, шифруются и сохраняются исключительно локально в `localStorage`. Они отправляются на бэкенд только во временных заголовках запроса (`X-MBX-APIKEY` / `X-MBX-SECRET`) и никогда не сохраняются на сервере.
    *   **Серверный режим (для виджета):** Возможность настроить постоянный доступ через переменные окружения сервера (`.env`), что идеально подходит для создания публичного виджета или домашней панели мониторинга.
*   **🧩 Компактный Виджет (`/widget`):** Специальный минималистичный интерфейс, оптимизированный для вывода на дополнительных экранах, встраивания в Notion, Obsidian или использования в качестве Iframe.

## 🛠️ Архитектура и технические особенности

*   **Синхронизация времени (`time_offset`):** Binance API блокирует запросы при рассинхронизации системных часов сервера и клиента. Бэкенд автоматически калибрует смещение времени перед каждым запросом.
*   **Отказоустойчивость:** Если у используемого API-ключа отключены права на просмотр какого-то из разделов (например, кошелька Пополнения или Earn), приложение не падает, а выводит доступные балансы и мягко сообщает об отсутствии необходимых прав.
*   **Serverless-ready:** Проект полностью готов к развертыванию на платформе **Vercel** в качестве Serverless-функции с использованием файла конфигурации `vercel.json`.

## 🚀 Быстрый запуск

### 1. Локальный запуск (Flask)

Вам понадобится установленный **Python 3.8+**.

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. (Опционально) Создайте файл `.env` в корневой папке для автоматической авторизации без использования настроек в браузере:
   ```env
   BINANCE_API_KEY=ваш_api_key
   BINANCE_API_SECRET=ваш_api_secret
   ```

3. Запустите веб-сервер:
   ```bash
   python app.py
   ```

4. Откройте приложение в браузере:
   *   Главный дашборд: [http://localhost:8000/](http://localhost:8000/)
   *   Мини-виджет: [http://localhost:8000/widget](http://localhost:8000/widget)

### 2. Деплой на Vercel

Проект оптимизирован под Vercel CLI. Для деплоя выполните в консоли:
```bash
vercel
```

## 🔑 Создание API-ключей в Binance

Для безопасной работы приложения:
1. Перейдите в аккаунте Binance во вкладку **Управление API** (API Management).
2. Создайте новый API-ключ ("Сгенерированный системой").
3. Пройдите проверку безопасности и скопируйте ключи.
4. **Важные ограничения (API Restrictions):**
   *   Достаточно включить галочку **«Включить чтение»** (Enable Reading).
   *   Все торговые функции, маржинальные операции и права на вывод средств (**Withdrawal**) **должны быть отключены**!
